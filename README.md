# Just Fruits
Collect as many fruits as possible in 60 seconds.
This was my first JavaScript experiment. It was created between 2008 and 2009.

[Try demo!](http://llinares.github.com/just-fruits/)

## Instructions
* W: Up
* S: Down
* A: Left
* D: Right
